# Contributing to Energy Intelligence Node

Welcome! We are building an open-source tool to protect communities from climate tech greenwashing.

## How to Contribute

### Identify New Data Sources
We need scrapers for regional energy databases (e.g., ANEEL in Brazil, CREG in Colombia).

### Improve Physics Regex
Help us extract more accurate metrics from complex scientific papers.

### Localization
Help us translate reports and interfaces into Spanish, Portuguese, and indigenous languages.

## Code of Conduct

Be respectful. Focus on physical evidence, not corporate marketing. We prioritize contributions that increase accessibility for underfunded institutions and MAPA communities.
